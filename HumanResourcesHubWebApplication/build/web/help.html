Help Page

This platform is designed for Human Resource Managers to manage employee records securely and efficiently.

Managers must log in with verified credentials. Incorrect login attempts are denied. New managers can register and must safeguard their login details.

After login, managers access the dashboard with the following tools:

  1. About – Overview and background of the web application.
  2. Add Employee – Add new employee records.
  3. Dashboard – Access core HR tools.
  4. Delete Employee – Remove employees from the database.
  5. Help – Guidance and support (this page).
  6. Home – Return to the welcome page.
  7. Log In – Access with manager credentials.
  8. Register – Register new manager credentials.
  9. Search Employee – Find employees using their staff number.
  10. Update Employee – Edit employee information.
  11. Read Employee – View all employees in the database.

Each page is designed for full functionality and ease of use.
