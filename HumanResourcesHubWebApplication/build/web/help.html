This is a website designed for Human Resource Managers for the Employees. It has all the details of of the managers
who exists in the site. If the manager enters the incorrect login details, he/she will be denied to login.
We have designed the system in such a way that only the managers with the correct login details, are allowed access.

If there's a new manager in the system, he/she will be given an opportunity to register and become part of the web app.
It is required and very much important that the managers never forget their login details and thus advised to write
them down.
After all the managers who exist in the system have logedIn, they are taken to the Dashboard.
They will be warmly greeted and given the below options of what they'll like to do today.

  1. About page explains the history behind our web app.
  2. Add Employee page allows the manager to persist new employees into the database.
  3. Dashboard page allows you to see all the options the manager wants to take in the website
  4. Delete Employee page removes the employees details in the database.
  5. Help ! is the page you have opened. It guides the user on how to use the web app and provides
    additional explanation you some concepts.
  6. Home page takes you to the welcome page
  7. LogIn page allows the manager to log into the web app.
  8. Register page allows new managers to register their information into the database.
  9. Search Employee page allows the manager to search for a specific employees using the employee staff number.
  10. Update Employee page updates the details of the employees.
  11. Read Employee page views all the employees persisted into the database.

Each and every page functions fully.
